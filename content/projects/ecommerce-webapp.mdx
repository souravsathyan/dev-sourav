# E-Commerce Website

This e-commerce platform is built to offer a complete online shopping
experience. It includes features such as product browsing, user authentication,
secure payment processing, and a responsive design that ensures usability across
devices. The website is built with technologies like Node js , Express, Html,
CSS, Js, Bootstrap.

## Technologies Used

The development of this e-commerce website involved the use of various
technologies and tools, including:

- **Node.js & Express:** Backend for managing API routes and handling
  server-side operations.
- **MongoDB:** NoSQL database to store user data, product information, and
  orders.
- **Razorpay API:** Integrated for handling secure online payments.
- **JWT:** Used for user authentication and managing login sessions securely.
- **Bootstrap:** A utility-first CSS framework for creating responsive and
  modern user interfaces quickly.

## Key Features

This e-commerce platform is equipped with the following features:

- **User Authentication:** Secure sign-up and login functionality using OAuth
  providers like Google and Facebook.
- **Product Management:** Admin dashboard for adding, updating, and managing
  product listings, including price, inventory, and categories.
- **Shopping Cart:** Users can add items to their cart, modify quantities, and
  proceed to checkout.
- **Payment Gateway Integration:** Seamless payment processing through Razorpay,
  ensuring secure transactions.
- **Order Tracking:** Users can view their order history and track the status of
  current orders.
- **Responsive Design:** Optimized for both desktop and mobile devices,
  providing a smooth shopping experience on any platform.
- **Search and Filtering:** Advanced search and filter functionality to help
  users find products easily based on categories, price range, or popularity.
