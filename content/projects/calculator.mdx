# Calculator

A fully functional calculator built using React and TypeScript. It features a simple and intuitive user interface, with support for basic arithmetic operations and more advanced functions such as exponentiation and logarithms.

## Technologies Used

The development of this calculator involved the use of various technologies and tools, including:

* **React:** For building reusable UI components and managing state effectively.
* **TypeScript:** For type safety and code maintainability.
* **Sass:** For styling the application with CSS.

## Key Features

This calculator is equipped with the following features:

* **Basic Arithmetic Operations:** Support for addition, subtraction, multiplication, and division.
* **Advanced Functions:** Support for exponentiation, logarithms, and trigonometric functions like sine, cosine, and tangent.
* **User-Friendly Interface:** A simple and intuitive interface for entering numbers and performing calculations.
* **Error Handling:** Proper error handling for invalid inputs and operations.
